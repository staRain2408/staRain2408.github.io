# PhotoTI 攝影人格測驗

這是一個一頁式攝影人格測驗網站。  
填完問卷後，會依照四組攝影偏好軸線計算出 16 種攝影人格。

## 使用方式

1. 建立一個新的 GitHub Repository
2. 上傳 `index.html`
3. 到 Repository 的 **Settings**
4. 左側選擇 **Pages**
5. Source 選擇 **Deploy from a branch**
6. Branch 選擇 `main`，資料夾選擇 `/root`
7. 儲存後等待 GitHub 產生網址

完成後網址通常會長得像：

```text
https://你的帳號.github.io/你的repository名稱/
```

## 檔案說明

- `index.html`：網站本體，包含 HTML / CSS / JavaScript
- `README.md`：說明文件

## 測驗軸線

- C / N：構築 Curated ｜ 自然 Natural
- S / A：主角 Spotlight ｜ 氛圍 Atmosphere
- P / F：表演 Perform ｜ 流動 Flow
- R / M：紀錄 Record ｜ 情緒 Mood
